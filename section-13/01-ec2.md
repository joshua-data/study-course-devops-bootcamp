# Login to the EC2 Instance. (web01 - dev)

```shell
# Connect to the instance.
chmod 400 ~/.aws/Moso-dev-seoul.pem
ssh -i ~/.aws/Moso-dev-seoul.pem ubuntu@43.201.84.135

# Install apache2.
apt update && apt install apache2 -y
systemctl status apache2

# Install the Moso template.
wget https://www.tooplate.com/zip-templates/2133_moso_interior.zip
apt install unzip -y
unzip 2133_moso_interior.zip
cp -r 2133_moso_interior/* /var/www/html/
systemctl restart apache2

# Check if apache2 is running.
ps -ef | grep apache2
ls /var/www/html/
ss -tunlp | grep 2545
```

### Finally, enter the url on the chrome browser.

```shell
http://43.201.84.135:80
```
